# 🎬 Movie App (React)

Bu proje, TMDB API kullanarak film detaylarını gösteren, fragman izleme imkanı sunan ve kullanıcı dostu bir arayüze sahip modern bir web uygulamasıdır.

## ✨ Özellikler
* **Fragman Desteği:** Filmlerin fragmanlarını modal içinde izleme.
* **Modern UI:** Responsive tasarım ve şık detay sayfaları.
* **Tema Desteği:** Karanlık ve Aydınlık mod seçeneği.
* **Watchlist:** İzlenecekler listesi yönetimi.

## 🚀 Teknolojiler
* React & Vite
* TMDB API
* Bootstrap & CSS
