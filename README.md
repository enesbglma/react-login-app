# React Login App 🚀

Vite ve React kullanılarak geliştirilmiş, Custom Hook yapısı ile güçlendirilmiş modern bir giriş formu uygulaması.

## Özellikler 🛠️
* **Custom Hooks:** Form yönetimi için özel `useInput` hook'u kullanıldı.
* **Validation:** E-posta formatı ve şifre uzunluğu (min 8 karakter) kontrolleri.
* **Context API:** Tema yönetimi (Dark/Light Mode) desteği.
* **Responsive Design:** Bootstrap ile mobil uyumlu arayüz.
